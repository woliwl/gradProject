2021年我的毕业设计--“庭好记”
===========================
设计的是一个家庭记账小程序，前端uniapp,后端thinkjs（nodejs的一类框架），数据库，mysql，实现个人账本。共享账本，功能账务增删查改、账务图表统计、家庭成员增删等已基本实现。后续会接着完善，需完善的地方有：\<br>  
1.ui设计\<br>  
2.实现对家庭的编辑、实现对家庭成员备注功能\<br>  
3.实现债务还款提醒功能\<br>  
4.优化交互等\<br>  
5.添加语音转文本功能，图片上传功能\<br>
